**定义  1. 1**    设 $A, B$ 为二集合，若 $B$ 中的每个元素都是 $A$ 中的元素，则称 $B$ 是 $A$ 的**子集**，也称 $A$ 包含 $B$ 或 $B$ 包含于 $A$，记作 $B \subseteq A$. 其符号化形式为 $B \subseteq A \Leftrightarrow \forall x(x \in B \to x \in A)$.
