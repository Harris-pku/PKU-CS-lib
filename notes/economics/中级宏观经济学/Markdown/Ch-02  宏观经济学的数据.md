#### 国民生产总值

**国民生产总值**：gross domestic product(GDP)



国民生产总值 = 消费 + 投资 + 政府购买 + 净出口

Y = C + I + G + NX

#### 失业率

**失业率**：the unemployment rate

失业率 $= \cfrac{失业人口}{劳动力人口}$ 

劳动参与率 $= \cfrac{劳动力人口}{成年人口}$
